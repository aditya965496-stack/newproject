# BRDGenAI

BRDGenAI is an AI-powered platform concept for generating Business Requirements Documents (BRDs) from fragmented communication channels such as Slack, Gmail, transcripts, and uploaded files.

## Repository contents

- `docs/PRD.md` – PRD-ready product specification.
- `index.html` – dashboard UI concept covering ingestion, extraction KPIs, editing workflow, and requirement snapshots.
- `styles.css` – responsive styles for the dashboard concept.
